# Smart Gym Exercise Recommendation System

The Smart Gym Exercise Recommendation System is a Machine Learning based project that recommends suitable exercises according to a user’s fitness profile and workout goals. The system analyzes factors such as age, gender, height, weight, BMI, workout duration, experience level, and available gym equipment to generate personalized exercise recommendations.

This project uses the Random Forest Classification algorithm to train the model on a gym fitness dataset and predict exercises such as cardio, yoga, HIIT, strength training, bodybuilding, and resistance training.

The project also includes Exploratory Data Analysis (EDA) using graphs and visualizations to understand fitness trends and user behavior. Technologies used in this project include Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

The system performs data preprocessing, label encoding, model training, evaluation, and prediction generation. The trained model is saved using Joblib for future use.
